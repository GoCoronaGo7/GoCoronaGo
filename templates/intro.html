Project - Gocoronago


This project was started in lieu with the Covid-19 pandemic. This website aims to allow users to
a ) show bed statistics and deatails of availability in nearbu hospitals
b ) allow for the users to post and view local covid related information via blog posts
c ) book appointments with doctors for a fee after logging in
d ) allow for doctors to accept and get in touch with patients requesting their services

This app was done by using Python ,Flask ,Jinja ,MySQL and a bit of ReactJS
