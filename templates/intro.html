Project - GoCoronaGo


This project was started in lieu with the Covid-19 pandemic. This website aims to allow users to
  • Show bed statistics and details of availability in nearby hospitals
  • Allow for the users to post and view local covid related information via blog posts
  • Book appointments with doctors for a fee after logging in
  • Allow for doctors to accept and get in touch with patients requesting their services
This project was made with Python, Flask, Jinja, PostgreSQL and ReactJS
